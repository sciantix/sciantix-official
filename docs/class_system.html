Parameters
input_valueThe model selection index for fission gas diffusivity.

setFissionGasDiffusivity

The intra-granular fission gas (xenon and krypton) diffusivity within the fuel grain is set according to the input_variable iFissionGasDiffusivity

iFissionGasDiffusivity = 0 corresponds to a constant intra-granular diffusivity value, equal to 7e-19 m^2/s.

iFissionGasDiffusivity = 1 set the fission gas (xenon and krypton) single-atom intragranular diffusivity equal to the expression in Turnbull et al (1988), IWGFPT-32, Preston, UK, Sep 18-22.

iFissionGasDiffusivity = 2 set the xenon effective intragranular diffusivity equal to the expression in Matzke (1980), Radiation Effects, 53, 219-242.

iFissionGasDiffusivity = 3 set the xenon single-atom intragranular diffusivity equal to the expression in Turnbull et al., (2010), Background and Derivation of ANS-5.4 Standard Fission Product Release Model.

iFissionGasDiffusivity = 4 set the xenon single-atom intragranular diffusivity equal to the expression in Ronchi, C. High Temp 45, 552-571 (2007).

Sets the helium diffusivity within the matrix based on the selected model. The intra-granular helium diffusivity within the fuel grain is set according to the input_variable iHeDiffusivity.

Parameters
input_valueThe model selection index for helium diffusivity.

setHeliumDiffusivity

The intra-granular helium diffusivity within the fuel grain is set according to the input_variable iHeDiffusivity

iHeDiffusivity = 0 corresponds to a constant intra-granular diffusivity value

iHeDiffusivity = 1 is the best-estimate correlation, from data available in literature, for samples with no or very limited lattice damage. This correlation is also recommended for simulations of helium in UO2 samples in which infusion technique has been adopted.

See also
The correlation is from L. Luzzi et al., Nuclear Engineering and Design, 330 (2018) 265-271.

iHeDiffusivity = 2 is the best-estimate correlation, from data available in literature, for samples with significant lattice damage. This correlation is also recommended for simulations of helium in UO2 samples in which implantation technique has been adopted.

See also
The correlation is from L. Luzzi et al., Nuclear Engineering and Design, 330 (2018) 265-271.

iHeDiffusivity = 2 sets the single gas-atom intra-granular diffusivity equal to the correlation reported in Z. Talip et al. JNM 445 (2014) 117-127.

iHeDiffusivity = 4 corresponds to a null intra-granular diffusivity value

Parameters

setProductionRate

Production rate = cumulative yield * fission rate density

Surrogate model derived from helium production in fast reactor conditions. The helium production rate is fitted with a function linearly dependent on the local burnup.

See also
The default fit is from A. Cechet et al., Nuclear Engineering and Technology, 53 (2021) 1893-1908.

Default range of utilization of the default fit

  • Fast reactor conditions: (U,Pu)O2 MOX fuel in SFR conditions
  • Up to 200 GWd/tHM
  • Pu/HM concentration of 20-40%

The default fit (hence the helium production rate) can be calibrated by using the dedicated scaling factor (to be set in input_scaling_factors.txt).

Constant production rate

Production rate = cumulative yield * fission rate density * HBS volume fraction
