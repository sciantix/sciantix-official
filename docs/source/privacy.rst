Privacy Policy for sciantixGPT
==============================

sciantixGPT is an AI assistant that provides read-only access to this public GitHub repository "sciantix/sciantix-official". 
It does not collect, store, or process personal data from users.

sciantixGPT does not write to the repository, does not access private information, 
and does not store conversation content or usage data outside the ChatGPT platform.

All interactions are processed by OpenAI according to the OpenAI Privacy Policy:
https://openai.com/policies/privacy-policy
