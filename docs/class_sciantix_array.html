Detailed Description

template<class T>
class SciantixArray< T >

A template class for managing a collection of variables with fast access by name or index.

Parameters
TThe type of elements stored in the array. The type T must have a getName() method returning a std::string.

This class allows for efficient storage, retrieval, and modification of elements by their name or index. It internally uses a std::vector for storing the elements and a std::map for mapping names to indices for quick lookup.
